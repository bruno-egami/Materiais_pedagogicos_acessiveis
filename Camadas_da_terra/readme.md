# Camadas da terra

Modelo de camadas da terra para impressão com matriz da legenda para uso em termoformadora.

O recurso conta com x arquivos, sendo que os arquivos em PDF devem ser impressos em folha A4. O arquivo com extensão dxf deve ser utilizado para o recorte da matriz.
Os arquivos stl´s são para impressão.
